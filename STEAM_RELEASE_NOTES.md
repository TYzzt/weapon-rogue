# 武器替换者 (Weapon Rogue) - Steam版发布说明

## 概述
《武器替换者》是一款roguelike游戏，核心机制是：敌人掉落的武器必定替换玩家当前的武器。这可能是好事，也可能是灾难...玩家需要通过策略性地选择和利用武器来生存并击败更多敌人。

## 核心功能

### 1. 武器替换机制
- 每次击败敌人必定掉落武器
- 新武器会替换当前装备的武器
- 不同稀有度的武器（普通、稀有、史诗、传说、神话）
- 武器有不同属性和特殊效果

### 2. 多样化敌人系统
- 基础敌人：近战、远程
- 精英敌人：高血量、高伤害
- Boss敌人：特殊技能和模式
- 扩展敌人类型：机械系、神话系、飞行系等

### 3. 技能系统
- Q键：旋风斩 - 对周围敌人造成伤害
- W键：治疗光环 - 恢复生命值
- E键：闪现 - 瞬移到鼠标位置
- R键：狂暴 - 短时间内大幅提升伤害

### 4. 连击系统
- 连续击杀敌人积累连击数
- 连击数越高，得分加成越大
- 保持连击需要不断击杀敌人

## 扩展功能

### 1. 存档系统
- localStorage保存游戏进度
- 自动备份和恢复功能
- 版本兼容性支持
- 包含成就、统计、设置等完整数据

### 2. 成就系统 (35+ 成就)
- 进度类：达到指定关卡
- 战斗类：击杀数量、连击数、得分等
- 武器类：使用不同武器、获得稀有武器
- 挑战类：特殊挑战成就

### 3. 音效系统
- 20+种游戏音效 (战斗、拾取、环境等)
- 9+种背景音乐 (适配不同游戏场景)
- Web Audio API支持，兼容性回退
- 音效/音乐独立控制

### 4. 菜单系统
- 主菜单：开始游戏、继续、设置、成就、教程
- 暂停菜单：继续、重启、设置、成就、主菜单
- 设置菜单：音效、音乐、难度、语言、手柄支持

### 5. 教程系统
- 6步完整新手引导
- 条件触发，适应玩家进度
- 中英文本地化

### 6. 特效系统
- 粒子系统：撞击、治疗、升级、武器拾取等
- 高级视觉效果和动画
- 战斗日志系统

### 7. 控制器支持
- Gamepad API完整支持
- 摇杆控制移动和瞄准
- 按钮映射技能和游戏功能

### 8. 本地化
- 中英文完整支持
- 实时语言切换
- 全界面文本翻译
