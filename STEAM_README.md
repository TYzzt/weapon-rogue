# 武器替换者 - Weapon Rogue (Steam版)

一款独特的 Roguelike 游戏，核心机制是：**敌人掉落的武器必定替换你当前的武器**。这可能是好事，也可能是灾难...收集药水和遗物来改变命运！

## 🎮 游戏特色

### 🗡️ 独特武器替换机制
- 每次获得新武器都会**强制替换**当前武器
- 紧张刺激的装备管理体验
- 每次拾取都是冒险，可能获得神装，也可能遭遇劣质武器

### 👾 丰富的敌人系统
- 30+种不同类型的敌人
- 近战、远程、精英、Boss 四种敌人类型
- 智能AI，根据血量和距离调整行为策略

### 🧪 多样化道具系统
- 药水系统：生命药水、武器保护剂、幸运药水、力量药水
- 遗物系统：贪婪护符、记忆水晶、时间沙漏等多种遗物
- 特殊效果：燃烧、冰冻、眩晕、中毒等元素效果

### ✨ 技能系统
- Q(旋风斩) - 大范围伤害
- W(治疗光环) - 持续恢复生命
- E(闪现) - 瞬移躲避攻击
- R(狂暴) - 伤害和攻速提升

### 🔥 连击与成就
- 连击系统：连续击杀可积累连击数，获得额外分数
- 40+个成就，提供长久的重玩价值
- 高分榜系统，挑战最高纪录

## 🚀 Steam版新增功能

### 🎯 核心玩法优化
- **智能敌人AI**：敌人会根据血量决定撤退，根据距离调整追击策略
- **生命恢复机制**：在安全环境下玩家可自然恢复生命
- **特殊效果武器**：包含燃烧、冰冻、眩晕等元素效果的武器
- **增强连击系统**：每10连击给予伤害加成，每50连击给予生命恢复

### 💾 完整游戏系统
- **高级存档系统**：支持进度保存、统计信息、设置选项
- **Steam版成就系统**：40+个精心设计的成就
- **完整音效系统**：30+种游戏音效，10+首背景音乐
- **多语言支持**：支持中英文界面切换

### 🎨 视觉与体验增强
- **精灵图支持**：精美图像替代原始色块
- **特殊效果**：元素武器对应的视觉反馈
- **完整菜单系统**：主菜单、暂停菜单、设置菜单、成就菜单
- **优化UI/UX**：直观的用户界面和流畅的操作体验

## 🎮 游戏操作

- 🖱️ **移动**：鼠标控制角色移动
- 🖱️ **攻击**：点击鼠标攻击范围内的敌人
- ⌨️ **技能**：Q/W/E/R 使用四个主动技能
- 🎒 **拾取**：点击掉落物自动拾取

## 🏆 成就列表 (部分)

- **初出茅庐**：获得你的第一滴血
- **连击之王**：达成10连击
- **武器收藏家**：使用过10种不同的武器
- **元素大师**：使用过5种不同元素效果的武器
- **生存专家**：达到第40关且生命值仍然大于70
- **武器鉴赏家**：使用过50种不同的武器
- 更多精彩成就等你解锁！

## 🎯 开发理念

我们致力于创造一款简单易懂但深度有趣的Roguelike游戏。武器替换机制带来了独特的策略元素 - 你永远不知道下一个武器是天使还是魔鬼。游戏在保持挑战性的同时，也提供了足够的辅助手段让玩家有翻盘的机会。
