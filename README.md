# 武器替换者 - Weapon Rogue

一款 roguelike 游戏，核心机制是：**敌人掉落的武器必定替换你当前的武器**。这可能是好事，也可能是灾难...

## 特色功能

- 🗡️ **武器替换机制**：每次获得新武器都会强制替换当前武器
- 👾 **多种敌人类型**：近战、远程、精英、Boss 四种敌人类型
- 🧪 **药水系统**：生命药水、武器保护剂、幸运药水、力量药水
- 🏺 **遗物系统**：贪婪护符、记忆水晶、时间沙漏等多种遗物
- ✨ **技能系统**：Q(旋风斩)、W(治疗光环)、E(闪现)、R(狂暴)
- 🔥 **连击系统**：连续击杀可积累连击数，获得额外分数

## 新增功能 (v1.1)

- 🖼️ **精灵图支持**：现在使用图像精灵替代原始色块
  - 武器使用 `assets/sprites/weapon_sword.png`
  - 史莱姆敌人使用 `assets/sprites/enemy_slime.png`
  - 其他敌人使用 `assets/sprites/enemies.png`
  - 药水使用 `assets/sprites/potions.png`
- 🔄 **动态图像加载**：实现预加载系统确保流畅体验
- 💥 **图像回退机制**：在图像加载失败时回退到原始色块

## 新增功能 (v1.2)

- ⚖️ **游戏平衡性增强**：优化武器稀有度权重分配，使用对数曲线调整不同等级的稀有度概率
- 🧩 **敌人系统扩展**：新增4种高级敌人类型，调整敌人生成权重确保多样性和挑战性
- 🗡️ **武器库扩充**：新增17种武器，覆盖所有稀有度级别
- 🧠 **AI行为增强**：实现敌人智能躲避和动态行为调整机制
- 📊 **玩家成长优化**：改进升级曲线，使游戏节奏更加合理
- 🎮 **控制器支持**：完整的Gamepad API支持
- 🌐 **国际化**：完整的中英文本地化支持
- 🏆 **成就系统**：30+个成就，完善的进度追踪
- 📚 **教程系统**：新手引导和游戏机制介绍
- 🎵 **音效系统**：背景音乐和各种游戏音效
- 💾 **存档系统**：localStorage保存游戏进度和设置
- ✨ **特效系统**：粒子系统和视觉特效增强

## 游戏操作

- 🖱️ **移动**：鼠标控制角色移动
- 🖱️ **攻击**：点击鼠标攻击范围内的敌人
- ⌨️ **技能**：Q/W/E/R 使用四个主动技能
- 🎒 **拾取**：点击掉落物自动拾取

## 项目结构

```
game-rogue/
├── index.html          # 游戏主页面
├── game.js             # 游戏核心逻辑
├── style.css           # 游戏样式
└── assets/
    └── sprites/        # 游戏精灵图
        ├── weapon_sword.png
        ├── enemy_slime.png
        ├── enemies.png
        └── potions.png
```
